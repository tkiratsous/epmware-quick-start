# EPMware Quick Start Guide

Welcome to the **EPMware Quick Start Guide**. This guide contains the high-level steps required to create and configure your first EPMware application. For detailed instructions on performing each step, please refer to the [Administrator's Guide](https://docs.epmware.com/admin-guide).

## About This Guide

This Quick Start Guide provides a streamlined path to get your EPMware application up and running quickly. Follow the steps in sequence to ensure proper configuration.


## Quick Setup Overview

The following high-level steps are required to configure your first EPMware application:

1. **[Add a Target Application Server](configuration/#add-target-application-server)** - Configure connection to your target systems
2. **[Create an EPMware Application](configuration/#create-epmware-application)** - Set up your first application
3. **[Change Security for Default Admin User](security/#default-admin-user)** - Secure your installation
4. **[Update Global Settings](global-settings/)** - Configure email and application settings
5. **[Start Services](services/#starting-services)** - Activate required services
6. **[Create a Workflow](workflow/#create-workflow)** - Design your approval process
7. **[Create a Deployment](deployment/#create-deployment)** - Set up deployment configuration

## Prerequisites

Before beginning the setup process, ensure you have:

- Administrator access to EPMware
- Target application server credentials
- Email server configuration details
